# Create-Badminton-Table

## 始めるとき
1. クローン
```
git clone "https://github.com/tomohiko9090/Create-Badminton-Table.git"

```

## 開発手順
1. branchを作り移動する
```
git checkout -b ブランチ名
```
注意！  
この時必ず、masterブランチからブランチを切ること！

2. コードを書く

3. pushを行う
```
git add .
git commit -m "何の機能を加えたか"
```

4. pull requestを行う
GitHubを用いてプルリクします。

5. 葛葉がブランチをmerge
