# Create-Badminton-Table
リーグ表作成アプリ<b>

## 一言でいうと...
- 試合（バドミントン, テニス, 卓球）の順番や組み合わせを考える手間を省けるアプリ

## 使用した技術
### 言語
- Ruby 2.7.6 
- JavaScript

### フレームワーク
- Sinatra

### データベース
- postgresql 14.4

### サーバー
- Heroku
